# 🎮 Reflex Click Game

A browser-based reflex game with leaderboard using Node.js.

## Features
- Timer-based gameplay
- Score tracking
- Backend API
- Leaderboard

## Tech Stack
HTML, CSS, JavaScript, Node.js

## How to Run

1. Install Node.js
2. Run:
   npm install
   node server.js
3. Open frontend/index.html
